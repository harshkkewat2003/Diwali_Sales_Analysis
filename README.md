# Diwali_Sales_Analysis


## 📌 Project Overview
This project analyzes customer purchasing behavior during Diwali festival sales using Python data analysis techniques.

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📊 Dataset
Diwali sales dataset containing customer demographics, product category, and purchase amount.

## 🔎 Steps Performed
- Data Cleaning
- Handling Missing Values
- Exploratory Data Analysis
- Data Visualization
- Business Insights Generation

## 📈 Key Insights
- Married women aged 26–35 are major buyers.
- Electronics category generated highest revenue.
- Most purchases came from Tier-1 cities.

## ✅ Conclusion
Businesses should target young working professionals and electronics promotions during festive seasons.
